Each milestone has its own `Prompt.md`, `Plan.md`, `Implement.md`, and `Documentation.md` so v26.3.0-alpha can be delivered incrementally without mixing scope.

## Order
M1-DET -> M2-OPC -> M3-VMSEC -> M4-AF -> M5-TL -> M6-DOC

- `M2-OPC/` - Opcode classification metadata + coverage checks
- `M3-VMSEC/` - Deterministic hooks + boundary enforcement + audit events
- `M4-AF/` - Artifact determinism metadata + replay proofs
- `M5-TL/` - Replay validator + deterministic CLI UX
- `M6-DOC/` - End-to-end release gate + docs
